# Bounce Ball Game
The goal of this game is to keep the ball in the ar.

![Bounce](Bounce.JPG)

## Level of Difficult
You can choose as many balls as you want. Just create a new ball class

## Actions
- `arrow to the right`
- `arrow to the left`
